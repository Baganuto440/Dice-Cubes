## 🎯 О проекте

**Dice & Cubes** — это интернет-магазин, специализирующийся на продаже дайсов (игровых кубов), книг правил и аксессуаров для Dungeons & Dragons.

**Основные пользователи:**
- Игроки в D&D и другие настольные RPG
- Мастера подземелий (Dungeon Masters)
- Коллекционеры игровых аксессуаров

**Решаемые задачи:**
- Удобный поиск и фильтрация игровых кубов
- Визуальная демонстрация товаров
- Оформление заказов
- Администрирование каталога

---

## 🛠 Технологии

### Frontend
- **HTML5** — семантическая разметка
- **CSS3** — адаптивный дизайн, анимации, тёмная тема
- **JavaScript (ES6)** — логика загрузки, интерактивность
- **MySQL / PostgreSQL** — база данных

### Backend (планируется)
- **PHP / Node.js** — серверная логика

### Деплой
- **Vercel** — хостинг статики

## ⚡ Функциональность

### Реализовано:
- ✅ Анимированный загрузочный экран с 3D-кубом
- ✅ Прогресс-бар с отображением процентов загрузки
- ✅ Тёмная тема в стилистике D&D
- ✅ Интерактивные элементы (ускорение куба при наведении)
- ✅ Плавный переход на основную страницу
- ✅ Модульная структура проекта (HTML/CSS/JS разделены)

### Планируется:
- 🔲 Каталог товаров с фильтрацией
- 🔲 Корзина и оформление заказов
- 🔲 Административная панель
- 🔲 Регистрация и авторизация пользователей
- 🔲 Интеграция с платежной системой
